# Service Agency Cogency Skills

This `skills.md` consolidates team-facing capabilities for **Mom's teamAET GSOC / AHJ / Drill Coordinator** overlays and aligned service operations.

## 1) Core Systems Translation
- Convert abstract proto-systems into plain-language playbooks.
- Map symbolic frameworks (`Init.0`, F-Gate, INNER·INTRA·INTER) into deployable milestones.
- Keep a two-variable operating mode for rapid scenario swapping:
  - `x = mission context`
  - `y = response tempo`

## 2) Queue + Workflow Orchestration
- Design and operate queue stages:
  - `answers` (intake)
  - `QZS` (zero-state triage)
  - `QTD` (task distribution)
  - `QDT` (delivery + telemetry)
- Provide handoff checkpoints for human operator + autonomous agent teams.

## 3) Hyperlinked Marketing Documentation
- Build sectioned documentation with:
  - anchored navigation
  - hover definitions/tooltips
  - copy-ready snippets
  - deep links to domain endpoints and campaign pages
- Ensure docs are readable by engineering, operations, and executive teams.

## 4) Conversational Rubik Vocabulary Swap
- Use two dynamic variables (`x`, `y`) to generate response variants:
  - executive brief
  - operator checklist
  - client-facing marketing narrative
  - emergency drill script
- Maintain semantic consistency while changing tone and delivery speed.

## 5) Desktop Overlay Expression
- Present key states in overlay cards:
  - coherence meter
  - queue depth
  - priority hazards
  - persona / role filter state
- Keep visual language stable across incident and demo modes.

## 6) Agent Coordination Skills
- Plan → Act → Verify loops for each run cycle.
- D&D action economy mapping for constrained turn-based execution:
  - Action = primary workflow step
  - Bonus Action = automation assist
  - Reaction = interrupt / alert handling

## 7) Reliability + Provenance
- Preserve provenance rows for every major artifact update.
- Track confidence and uncertainty notes for generated insights.
- Add explicit "AI can make mistakes" review callouts in externally shared docs.

## 8) Deployment Readiness
- Maintain environment matrix for domains and backend ports (`8000-9000`).
- Include bootstrap sequence and warning labels for partial DNS states.
- Support one-command dry-run validation prior to launch.

## 9) Drill Coordinator Playbooks
- Auto-generate drill scripts from queue state and role roster.
- Produce after-action summaries with links back to source evidence.
- Provide concise next-step recommendations for municipal partners.
